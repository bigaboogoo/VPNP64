🌐 VPNP64 - Web Proxy & Internet Archive
A powerful web-based proxy system that allows users to access any part of the internet 24/7, even when websites are down, by serving cached snapshots of the most-visited pages.

Features
✨ Intelligent Caching - Automatically caches frequently visited websites 🔒 Security - Built-in SSRF protection and private IP blocking ⚡ Fast Access - Serves cached pages instantly when available 🌍 Reverse Proxy - Support for both forward and reverse proxy modes 📊 Statistics Dashboard - Real-time cache hit/miss tracking 🎨 Modern UI - Clean, dark-themed web interface 🔄 Asset Handling - Automatically processes and serves related assets ⏱️ Smart Fallback - Serves stale cache when site is unreachable
